Functionality

Managing of applications for available jobs

The system is able to manage application (supporting basic CRUD-Operations) and notifies the employer when a new application is submitted as well as the applicant when his application is accepted.

All notifications are sent using SMTP.

Filtering of available jobs

The system receives a set of available jobs and filters them returning a sorted feed according to the user's parameters.
